# Fantasy Baseball Auction Calculator - Documentation

## Project Overview

A production-grade React single-page application for fantasy baseball auction draft value calculation and live draft management. Features a sophisticated calculation engine with Z-score analysis, position allocation, and real-time inflation tracking.

### Quick Reference

| Attribute | Value |
|-----------|-------|
| **Type** | Web Application (Monolith) |
| **Primary Language** | TypeScript |
| **Framework** | React 18 + Express.js |
| **UI Library** | Shadcn/ui (Radix primitives) |
| **Styling** | Tailwind CSS |
| **State** | React Context + localStorage |
| **Build Tool** | Vite |

### Entry Points

| Route | Page | Description |
|-------|------|-------------|
| `/` | League Settings | Configuration wizard (league, scoring, projections, values) |
| `/draft` | Draft Room | Live draft interface with player table and metrics |

## Getting Started

### Prerequisites
- Node.js v20.x+
- npm v10.x+

### Quick Start
```bash
npm install
npm run dev
# Open http://localhost:5000
```

### Application Flow
1. **Configure League** - Set team count, budget, roster positions
2. **Select Scoring** - Choose Roto, H2H Categories, or H2H Points
3. **Upload Projections** - Import hitter/pitcher CSVs
4. **Calculate Values** - Generate auction values using Z-Score method
5. **Draft** - Track picks with real-time inflation adjustments

## Key Features

### Calculation Engine (v2.0)
- Per-position Z-score analysis with rate stat weighting
- Multi-eligibility handling (MI, CI, UTIL, P slots)
- Three replacement level methods (Last Drafted, First Undrafted, Blended)
- VAR (Value Above Replacement) calculation
- Configurable hitter/pitcher budget split
- Position scarcity multipliers (optional)
- Value tiers: Elite, Star, Starter, Bench, Replacement

### Draft Room
- Real-time inflation tracking
- Pending bid system (prices "locked" before confirmation)
- Quick draft interface
- Pick history with undo
- Positional needs tracker
- PDF cheat sheet export

### Data Persistence
- All state persisted to browser localStorage
- No backend database required
- Automatic save on every change
